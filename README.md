<strong><h2>Project Conclusion</h2></strong>
On 5/14/2026, the core features of the to-do list web app have been added. As I stated, this is a personal project. The point of this project was not to develop a fancy to-do list app, but rather, to practice my Front-end development skills, learn new coding solutions, and get used to the web development workflow as it would be seen in a professional workplace. Neither did I plan to add any fancy/fun features, nor did I attempt to make it outstanding. That was not the intent of this app. I kept the project's scope very small so that I didn't end up introducing unnecessary features that would increase the intended development time. The reason is so that I can learn from this project and begin a new project focusing on a concept that I may have struggled with. Through this project, I was able to learn more about the event object and how to get a hold of an HTML element effectively and manipulate it through JavaScript. The event object is something that I do want to focus on with my next project. The part I enjoyed the most was developing the task deletion function. This was the function where I had to use the event object to get the textContent of the p element within a section element that housed the task data. I then used that textContent to conduct a search through an array of objects to search for the object responsible for the clicked tasks section element, and then delete it when the user clicks on that specific task. It took some time, but with some brainstorming, I was able to create a fully functional app. I'm happy with how it has turned out so far, and I will be moving on to planning my next project.
